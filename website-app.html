Install prompt available

Your browser supports app installation. Use the button below to install Semitic Jew as a website app.

iPhone / iPad

Install on iOS

  1. Open this site in Safari.
  2. Tap the Share button.
  3. Select Add to Home Screen.
  4. Launch Semitic Jew from your Home Screen.
Android / Desktop

Install on Chrome or Edge

  1. Open this site in Chrome or Edge.
  2. Tap the browser menu or install icon.
  3. Select Install App or Add to Home screen.
  4. Launch it like a normal app.
Why Install

Faster access to the study ecosystem.

The website app keeps Scripture, articles, study resources, media, and dietary tools one tap away without waiting for a native app release.
